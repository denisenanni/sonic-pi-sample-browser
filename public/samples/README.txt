Sonic Pi Sample WAV Files
=========================

Place all Sonic Pi built-in WAV files in this directory.

The samples are located in your local Sonic Pi installation:

  macOS:   /Applications/Sonic Pi.app/Contents/Resources/samples/
  Linux:   /usr/share/sonic-pi/samples/
  Windows: C:\Program Files\Sonic Pi\app\server\native\samples\

Copy all *.wav files from that directory into here (public/samples/).
File names must match exactly — e.g. bd_haus.wav, ambi_choir.wav, etc.

The WAV files are released under a permissive open-source licence and
are safe to redistribute alongside this project.
